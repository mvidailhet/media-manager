# Use Mantine for the React UI

We will use Mantine as the React UI component system for the Tauri frontend. The app needs practical desktop-style surfaces such as searchable video lists, filters, detail panels, batch selection, review queues, forms, badges, progress states, modals, and notifications; Mantine provides these with less custom styling work than building directly from primitives. This trades some visual/library lock-in for faster delivery of a clean local desktop UI.
